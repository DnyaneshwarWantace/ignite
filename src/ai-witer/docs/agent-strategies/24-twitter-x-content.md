# Twitter/X Content - Complete Strategy Guide

## Overview
Twitter/X Content creation focuses on engaging, educating, and selling through concise, valuable posts. Whether single tweets or threads, great Twitter content hooks attention, delivers value, and drives action without being pushy.

## Core Framework

### Content Types

#### Single Tweet
**Length:** 280 characters max (optimal: 100-150)

**Structure:**
- Hook (first line)
- Value/insight
- CTA (optional)

**Best Practices:**
- Lead with hook
- Deliver value fast
- Be concise
- Create engagement

#### Thread
**Length:** Multiple tweets (3-20 tweets)

**Structure:**
- Hook tweet
- Value tweets
- CTA tweet

**Best Practices:**
- Number the tweets
- Each tweet stands alone
- Build on previous
- Clear progression

## Content Formulas

### Formula 1: The Hook + Value
"[Hook] → [Value/Insight]"

**Example:**
"Most founders think they need to work harder.
The truth: You need to work smarter.
Here's how: [value]"

### Formula 2: The Contrarian
"Everyone says [common belief].
But here's why that's wrong: [insight]"

**Example:**
"Everyone says you need more leads.
But more leads = more unqualified prospects.
Better: Qualify BEFORE they reach sales."

### Formula 3: The Story
"I used to [struggle].
Then I discovered [solution].
Now I [result].
Here's what changed: [insight]"

**Example:**
"I used to work 80-hour weeks.
Then I discovered the 80/20 rule.
Now I work 20 hours/week.
Here's what changed: I focus on high-impact activities only."

### Formula 4: The List
"[Number] [Things] That [Action] [Outcome]:"

**Example:**
"7 mistakes that kill your revenue growth:
1. [Mistake]
2. [Mistake]
3. [Mistake]
..."

### Formula 5: The Question
"[Provocative Question]?
Here's the answer: [insight]"

**Example:**
"Why do most businesses fail at scaling?
They scale the wrong things.
Here's what to scale instead: [insight]"

### Formula 6: The Statistic
"[Surprising Stat] → [Insight] → [Value]"

**Example:**
"80% of founders work on low-impact activities.
That's why revenue stays flat.
Focus on the 20% that drives 80% of results."

## Thread Structure

### Thread Formula
**Tweet 1:** Hook + Teaser
**Tweet 2-5:** Value/Insights
**Tweet 6-10:** Examples/Case Studies
**Tweet 11-15:** Action Steps
**Final Tweet:** CTA + Engagement

### Thread Best Practices
- Number each tweet (1/10, 2/10, etc.)
- Each tweet readable alone
- Build on previous tweets
- Clear progression
- Strong conclusion

## Best Practices

1. **Be Authentic** - Real voice, not corporate
2. **Provide Value** - Every tweet should add value
3. **Engage Actively** - Respond, interact, build community
4. **Be Consistent** - Regular posting schedule
5. **Test and Learn** - Try different approaches

## Common Mistakes

1. **Too Salesy** - Pushing instead of providing value
2. **No Value** - Wasting followers' time
3. **Inconsistent** - Irregular posting
4. **No Engagement** - Not interacting
5. **Generic** - Not standing out
