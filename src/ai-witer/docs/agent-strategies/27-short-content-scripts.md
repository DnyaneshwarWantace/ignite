# Short Content Scripts - Complete Strategy Guide

## Overview
Short Content Scripts create specialized content in list formats (tips, steps, myths, mistakes, etc.). These scripts are perfect for social media, videos, and quick-value content that educates and engages.

## Core Framework

### List Format Types

#### Type 1: Tips List
**Structure:** "[Number] Tips to [Achieve Outcome]"

**Example:**
"7 Tips to 10X Your Revenue:
1. [Tip with explanation]
2. [Tip with explanation]
..."

**Best Practices:**
- Actionable tips
- Specific and clear
- Practical advice
- Easy to implement

#### Type 2: Steps List
**Structure:** "[Number] Steps to [Achieve Outcome]"

**Example:**
"5 Steps to Scale Your Business:
Step 1: [Action]
Step 2: [Action]
..."

**Best Practices:**
- Sequential steps
- Clear progression
- Actionable actions
- Logical order

#### Type 3: Mistakes List
**Structure:** "[Number] Mistakes That [Cause Problem]"

**Example:**
"7 Mistakes That Kill Your Revenue Growth:
1. [Mistake] - Why it's wrong: [explanation]
2. [Mistake] - Why it's wrong: [explanation]
..."

**Best Practices:**
- Common mistakes
- Why they're wrong
- How to avoid
- Better alternatives

#### Type 4: Myths List
**Structure:** "[Number] Myths About [Topic] Debunked"

**Example:**
"5 Myths About Scaling Your Business:
Myth 1: [Belief]
Reality: [Truth]
..."

**Best Practices:**
- Common myths
- Why they're wrong
- Real truth
- Proof/evidence

#### Type 5: Signs List
**Structure:** "[Number] Signs You [Need/Have] [Situation]"

**Example:**
"10 Signs You're Ready to Scale:
1. [Sign]
2. [Sign]
..."

**Best Practices:**
- Clear indicators
- Specific signs
- Actionable insights
- Helpful guidance

#### Type 6: Ways List
**Structure:** "[Number] Ways to [Achieve Outcome]"

**Example:**
"8 Ways to Generate More Qualified Leads:
1. [Method]
2. [Method]
..."

**Best Practices:**
- Diverse methods
- Practical approaches
- Clear explanations
- Actionable advice

## Script Structure

### Opening Hook
**Purpose:** Grab attention

**Formats:**
- Surprising statement
- Question
- Statistic
- Bold claim

**Example:**
"Most founders make these 7 mistakes that kill their revenue growth..."

### List Introduction
**Purpose:** Set up the list

**Elements:**
- List topic
- Why it matters
- What they'll learn
- Value promise

**Example:**
"Here are 7 mistakes that prevent you from scaling, and how to avoid them..."

### List Items
**Purpose:** Deliver value

**Structure per Item:**
- Number/Title
- Explanation
- Example (optional)
- Action (optional)

**Example:**
"1. Working on low-impact activities
Most founders spend 80% of time on tasks that drive 20% of results.
Instead: Focus on the 20% that drives 80% of results."

### Closing
**Purpose:** Summarize and CTA

**Elements:**
- Key takeaway
- Action step
- CTA (optional)
- Engagement ask

**Example:**
"The key: Focus on high-impact activities only.
Which of these mistakes are you making?
Comment below..."

## Content Optimization

### Length Guidelines
- **Social Media:** 5-10 items
- **Blog Posts:** 10-20 items
- **Videos:** 7-15 items
- **Emails:** 5-7 items

### Formatting
- Numbered lists
- Bullet points
- Sub-bullets for details
- Bold key points
- White space

### Visual Elements
- Graphics for each point
- Icons/emojis
- Charts/diagrams
- Images

### Step 3: Create Content
- Write hook

### Step 4: Optimize
- Edit for clarity
- Add visuals
- Format properly

## Best Practices

1. **Be Specific** - Clear, actionable items
2. **Provide Value** - Real insights, not fluff
3. **Use Examples** - Make it concrete
4. **Keep It Scannable** - Easy to read
5. **Include CTAs** - Guide to action

## Common Mistakes

1. **Too Generic** - Vague, not helpful
2. **No Value** - Doesn't help
3. **Too Long** - Overwhelming
4. **Poor Formatting** - Hard to read
5. **No CTAs** - Misses opportunity
