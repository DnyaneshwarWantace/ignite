# Viral Scripts - Complete Strategy Guide

## Overview
Viral Scripts create video scripts following proven structures from the world's most viral videos. These scripts combine attention-grabbing hooks, engaging content, and strong calls-to-action to create videos that get shared and drive results.

## Core Framework

### Viral Script Structure

#### Hook (First 3-5 Seconds)
**Purpose:** Stop the scroll and grab attention

**Elements:**
- Pattern interrupt
- Surprising statement
- Provocative question
- Bold claim

**Best Practices:**
- Immediate impact
- Create curiosity
- Stand out
- Promise value

#### Setup (Seconds 5-15)
**Purpose:** Build context and create intrigue

**Elements:**
- Context building
- Problem introduction
- Curiosity creation
- Value promise

**Best Practices:**
- Quick setup
- Maintain interest
- Build anticipation
- Create questions

#### Value Delivery (Seconds 15-45)
**Purpose:** Deliver the main value/insight

**Elements:**
- Core message
- Key insights
- Valuable information
- Actionable content

**Best Practices:**
- Clear value
- Specific insights
- Actionable advice
- Memorable points

#### Engagement (Seconds 45-60)
**Purpose:** Keep viewers watching

**Elements:**
- Story continuation
- Additional value
- Examples/case studies
- Visual interest

**Best Practices:**
- Maintain engagement
- Add depth
- Provide examples
- Visual variety

#### Call to Action (Final 5-10 Seconds)
**Purpose:** Drive desired action

**Elements:**
- Clear CTA
- Benefit reminder
- Urgency (if applicable)
- Next step

**Best Practices:**
- Clear and direct
- Benefit-focused
- Easy to complete
- Memorable

## Proven Viral Structures

### Structure 1: Problem-Solution-Result
**Format:**
1. Hook: Problem statement
2. Setup: Problem details
3. Value: Solution approach
4. Proof: Results/evidence
5. CTA: Get solution

**Duration:** 30-60 seconds

### Structure 2: Story Arc
**Format:**
1. Hook: Story opening
2. Setup: Context/background
3. Conflict: Problem/challenge
4. Resolution: Solution/outcome
5. CTA: Learn more

**Duration:** 45-90 seconds

### Structure 3: List Format
**Format:**
1. Hook: List promise
2. Setup: Why it matters
3. Value: List items (3-7)
4. Summary: Key takeaway
5. CTA: Action step

**Duration:** 30-60 seconds

### Structure 4: Contrarian Reveal
**Format:**
1. Hook: Contrarian statement
2. Setup: Common belief
3. Value: Real truth
4. Proof: Evidence
5. CTA: Learn more

**Duration:** 30-60 seconds

### Structure 5: Transformation Journey
**Format:**
1. Hook: Before state
2. Setup: The struggle
3. Value: The method
4. Proof: After state
5. CTA: Get transformation

**Duration:** 45-90 seconds

## Timing Guidelines

### Short-Form (15-60 seconds)
- Hook: 3-5 seconds
- Setup: 5-10 seconds
- Value: 20-30 seconds
- CTA: 5-10 seconds

### Medium-Form (60-180 seconds)
- Hook: 5-10 seconds
- Setup: 10-20 seconds
- Value: 60-120 seconds
- CTA: 10-15 seconds

### Long-Form (180+ seconds)
- Hook: 10-15 seconds
- Setup: 20-30 seconds
- Value: 120-180 seconds
- CTA: 15-20 seconds

## Best Practices

1. **Strong Hook** - First 3 seconds critical
2. **Quick Value** - Deliver value fast
3. **Visual Interest** - Keep it engaging
4. **Clear CTA** - Obvious next step
5. **Test and Optimize** - Measure what works

## Common Mistakes

1. **Weak Hook** - Doesn't grab attention
2. **Slow Start** - Takes too long to get to value
3. **No CTA** - Misses conversion opportunity
4. **Too Long** - Loses attention
5. **No Value** - Doesn't deliver
